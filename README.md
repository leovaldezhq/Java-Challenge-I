<h1>Desafio Java</h1>
<p>Implementar um app para controlar uma conta bancária</p>

<h2>Descrição</h2>
<p>
Este projeto simula uma conta bancária simples, permitindo que o usuário consulte o saldo,
realize transferências, receba valores e finalize a aplicação através de um menu interativo.
O programa é executado no terminal e utiliza a classe Scanner para leitura de dados
</p>

<h2>Orientações</h2>
<ul>
  <li>Criar um cabeçalho inicial com os dados do cliente (Nome, Tipo da Conta e Saldo)</li>
  <li>Criar um menu que descreve as operações. Aqui você pode escolher o nome de método que mais lhe agradar, como saca (ou transfere, enviaPix) para simular a retirada de valores da conta e deposita (ou recebeTransferencia, recebePix) para representar a entrada de valores na conta</li>
  <li>O menu deve ter quatro opções: a de entrada de valor, saída de valor, consulta de saldo e finalização da aplicação</li>
  <li>Lembre-se que para fazer a saída de valores, é necessário ter saldo suficiente</li>
  <li>O menu deve aparecer continuamente até que o usuário digite a opção para sair</li>
  <li>Caso ele digite qualquer opção que não seja correta, deve apresentar a mensagem de opção inválida</li>
  <li>Usaremos a classe <code>Scanner</code> para fazer a leitura da opção do usuário</li>
</ul>

<h2>Conceitos Trabalhados</h2>
<ul>
    <li>Estruturas de repetição</li>
    <li>Estruturas condicionais</li>
    <li>Criação e utilização de métodos</li>
    <li>Passagem de parâmetros e retorno de valores</li>
    <li>Uso da classe <code>Scanner</code> para entrada de dados</li>
</ul>

<h2>Como Executar</h2>
<ol>
    <li>Clone o repositório</li>
    <li>Compile o arquivo Java</li>
    <li>Execute a classe <code>Main</code></li>
    <li>Interaja com o menu pelo terminal</li>
</ol>
