<h1>Java Bank Account Challenge</h1>
<p>Implement an application to manage a terminal based bank account.</p>

<h2>Description</h2>
<p>
This project simulates a simple bank account, allowing the user to check the balance,
make transfers, receive funds, and close the application through an interactive menu.
The program runs in the terminal and uses the <code>Scanner</code> class to read user input.
</p>

<h2>Guidelines</h2>
<ul>
  <li>Create an initial header displaying the client’s information (Name, Account Type, and Balance);</li>
  <li>Create a menu describing the available operations. You may choose method names as you prefer, such as withdraw to simulate withdrawing funds from the account, and deposit to represent adding funds to the account;</li>
  <li>The menu must have four options: add funds, withdraw funds, check balance, and exit the application;</li>
  <li>Remember that to withdraw funds, there must be sufficient balance available;</li>
  <li>The menu should be displayed continuously until the user selects the option to exit;</li>
  <li>If the user enters an invalid option, an “invalid option” message should be displayed;</li>
  <li>Use the <code>Scanner</code> class to read the user’s menu option.</li>
</ul>

<h2>Concepts Covered</h2>
<ul>
    <li>Loop structures;</li>
    <li>Conditional structures;</li>
    <li>Creating and using methods;</li>
    <li>Passing parameters and returning values;</li>
    <li>Using the <code>Scanner</code> class for input handling.</li>
</ul>

<h2>How to Run</h2>
<ol>
    <li>Clone the repository.</li>
    <li>Compile the Java file.</li>
    <li>Run the <code>Main</code> class.</li>
    <li>Interact with the menu through the terminal.</li>
</ol>
